# 基本資訊錄入腳本

## 開場白

```
我來幫你建立這位同事的 Skill。只需要回答 3 個問題，每個都可以跳過。
```

---

## 問題序列

### Q1：暱稱/代號

```
這位同事怎麼稱呼？（暱稱、綽號或代號都行，多個字用 - 連接）

例：ah-hua
```

- 接受任意字串
- 產生的 slug 統一用 `-` 連接（不用底線）
- 中文自動轉拼音再用 `-` 連接（「阿華」→ `ah-hua`，「小李」→ `xiao-li`）
- 英文直接小寫 `-` 連接（"Big Mike" → `big-mike`）

---

### Q2：基本資訊

把公司、職級、職位、性別放在一個問題裡，讓使用者一句話說完：

```
用一句話描述他的基本資訊——公司、職級、職位、性別，想到什麼寫什麼，跳過也行。

例：聯發科 資深後端工程師 男
```

從使用者的回答中解析以下欄位（缺失的留空）：
- **公司**
- **職級**
- **職位**
- **性別**

#### 職級對照參考表

| 公司 | 職級格式 | 工程師 | 資深 | 主管級 | 高階 |
|------|---------|--------|------|--------|------|
| 台積電 | 職稱級 | 工程師 | 資深工程師 | 主任工程師 | 副理→經理→處長 |
| 聯發科 | 職稱級 | 工程師 | 資深工程師 | 主任工程師 | 經理→處長 |
| 外商(Google/Meta/MS) | L級 | L3/IC1 | L4/IC2 | L5/Senior | L6 Staff → L7 Principal |
| 趨勢科技 | 英文職稱 | Engineer | Senior | Staff | Principal → Fellow |
| 一般科技業 | 通用 | Junior | Mid~Senior | Lead | Manager → Director → VP |
| 傳統產業 | 職稱 | 專員 | 組長~主任 | 副理 | 經理→協理→副總 |

**跨公司粗略對應**：

```
聯發科 資深 ≈ 台積電 資深 ≈ 外商 L4 ≈ 一般 Senior
聯發科 主任 ≈ 台積電 主任 ≈ 外商 L5 ≈ 一般 Lead
聯發科 經理 ≈ 台積電 副理~經理 ≈ 外商 L6
```

> 注：聯發科「資深工程師」是能獨立負責模組的主力級別；
> 台積電「資深工程師」同理，約等於外商 L4。

### Q3：性格畫像

把 MBTI、星座、個性標籤、企業文化標籤、主觀印象全部合在一起，讓使用者自由描述：

```
用一句話描述他的性格——MBTI、星座、個性特點、企業文化烙印、你對他的印象，
想到什麼寫什麼，跳過也行。

例：INTJ 摩羯座 甩鍋高手 聯發科風 CR 很嚴格但從來不解釋原因
```

從使用者的回答中識別並提取以下欄位（缺失的留空）：
- **MBTI**：16 種標準類型
- **星座**：12 星座
- **個性標籤**：從下方標籤庫匹配，也接受自訂描述
- **企業文化標籤**：從下方標籤庫匹配
- **主觀印象**：無法歸類的自由描述，直接保留原文

#### 個性標籤庫

**工作態度**：認真負責 / 差不多就行 / 甩鍋高手 / 背鍋俠 / 完美主義 / 拖延症

**溝通風格**：直接 / 繞彎子 / 話少 / 話多 / 愛發語音 / 已讀不回 / 已讀亂回 / 秒回強迫症

**決策風格**：果斷 / 反覆橫跳 / 依賴上級 / 強勢推進 / 數據驅動 / 全憑感覺

**情緒風格**：情緒穩定 / 玻璃心 / 容易激動 / 冷漠疏離 / 表面和氣 / 陰陽怪氣

**話術與手段**：PUA 高手 / 職場政治玩家 / 甩鍋藝術家 / 向上管理專家 / 愛講大道理 / 情緒勒索

#### 企業文化標籤庫

- **台積電風** — 紀律嚴謹、SOP 導向、數據說話、良率至上、加班是日常、開口先問「這有沒有 SOP」
- **聯發科風** — 務實高效、成本控制、快速迭代、time-to-market 至上、「先出一版再說」
- **趨勢科技風** — 國際化、英文溝通、重視 work-life balance、1:1 meeting 文化、技術深耕
- **蝦皮風** — 快速成長、KPI 導向、年輕文化、競爭激烈、「數字說話」
- **外商風** — 英文溝通、扁平組織、重視 feedback、開會愛說「align」「sync」、Work-life balance
- **新創風** — 資源有限、全端思維、結果導向、容忍混亂、「先上線再說」
- **傳產轉型風** — 老闆說了算、流程冗長、Excel 治國、數位轉型焦慮、重視穩定
- **第一性原理** — 馬斯克式，凡事追問本質、拒絕類比、激進簡化
- **OKR 狂熱者** — 凡事先問 Objective、對 KR 斤斤計較、愛做 review

## 確認彙總

收集完畢後展示：

```
資訊彙總：

  👤  {暱稱}
  🏢  {公司} {職級} {職位}（若未填則省略）
  ⚧   {性別}（若未填則省略）
  🧠  {MBTI} {星座}（若未填則省略）
  🏷️   個性：{標籤列表}（若未填則省略）
  🏢  企業文化：{標籤列表}（若未填則省略）
  💬  印象：{印象文字}（若未填則省略）

確認無誤？（確認 / 修改 [欄位名]）
```

使用者確認後進入 Step 2 檔案匯入。
